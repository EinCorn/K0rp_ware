# K0rp_ware v0.2

První stabilní třímodulový řez.

Obsahuje:

- K0rp ClickAudit v0.2
- K0rp Fidget v0.2
- K0rp Bloom v0.2

Hlavní body:

- lokální desktop widgety
- webové verze v pultu
- pin / always-on-top ovládání
- lokální ukládání progresu tam, kde to dává smysl
- žádné účty
- žádný cloud sync
- žádná telemetrie

macOS buildy jsou unsigned vývojové buildy, takže první spuštění může vyžadovat běžné potvrzení přes Finder Open.
